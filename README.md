# Portfolio Ugo Massa

Bienvenue sur mon portfolio ! Ce projet contient mes fichiers de portfolio pour le BTS SIO SISR.

## Fichiers inclus
- `portfolio.htlm` : Version principale du portfolio
- `portfolio-ugo-massa.html` : Autre version du portfolio

## Comment utiliser
Ouvrez les fichiers HTML dans votre navigateur pour voir le portfolio.

## Mise en ligne
Ce dépôt est prêt à être poussé sur GitHub. Utilisez les commandes suivantes :

```bash
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/votre-utilisateur/portfolio.git
git push -u origin main
```

Remplacez `votre-utilisateur` par votre nom d'utilisateur GitHub.
